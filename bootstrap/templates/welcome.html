What is it and why?

It's a bunch of tools, totally written in Vanilla Javascript, designed to provide a basic template, variables and event processing using new HTML attributes. That means no Typescript nor Node/npm are required!

The result is lighter and faster projects with a significant reduced amount of dependencies! And all you have to learn is Vanilla Javascript!

Features

  • Template processing

    Example: 
                            
    <div id="ex1"
        replace_template='{
            "template":"/templates/examples/example1.html"
        }'
    >

    </div>

  • Event support

    It's refered to javascript events, but now, passing your modules and methods directly to Javascript runtime through some different attributes.

    Example: 
                            
    <div id='test' event_click='{
                                "action":"Mymodule.someMethod",
                                "params":{"p1": "This is a test!"}
                   }'>
        More HTML code here...
    </div>

  • Variable processing

    You can use variables on templates, off course! You need only to name one like this: 
                            
    ${myvar}
    Then, you can use a custom HTML attribute to replace this variable. 
                            
    <div id="ex" replace_vars='{"myvar": "This is a test!"}'>
        ${myvar}
    </div>
    You can call 'replace_vars' using Javascript as well. 
                             

    /* mymodule.js */

    import Base from '/static/js/base.js'


    export default class MyModule extends Base {

        constructor(app){
            super()
            this.app = app
            var vars = JSON.stringify({"myvar": "This is a test!"})
            var target = document.querySelector("#ex")
            this.app.action.replaceVars(target, target, vars)
        }
    }
  • Element repeating 
                            
    <div id="some_div" repeat=5>Repeat this</div>

Check out the documentation for more details about this and much more!
